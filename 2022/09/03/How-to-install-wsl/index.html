WSL安装教程（简化版）

2022/9/3 Linux

最近在学习linux，所以今天就出一个安装wsl的教程

具体的安装教程可以参考微软wsl安装文档
WSL就是适用于Linux的Windows子系统，可让开发人员直接在Windows上按原样运行GNU/Linux环境，且不会产生传统虚拟机或双启动设置开销。目前WSL有两个版本，WSL1 和 WSL2。
若要使用 WSL 2，需要运行 Windows 10 或者 Windows 11 。

  • 对于 x64 系统：版本 1903 或更高版本，内部版本为 18362 或更高版本。
  • 对于 ARM64 系统：版本 2004 或更高版本，内部版本为 19041 或更高版本。

    第一步

    我们需要进入bios将系统的VT(虚拟化功能)打开，打开VT的具体方法大家可以网上搜索，这里不过多赘述。

    第二步

    具体安装方法有两种。

    第一种方法（不推荐）

    直接用Windows的PowerShell或者cmd（这两个都要用管理员身份运行）运行下面这个代码（个人不推荐使用这种办法，因为安装过程会出现各种报错，并且用这种办法需要较新的系统版本）
    wsl --install

    第二种方法（推荐）

    步骤1

    用管理员身份打开PowerShell，然后运行下面这段代码：
    dism.exe /online /enable-feature /featurename:Microsoft-Windows-Subsystem-Linux /all /norestart
    如果你只想使用WSL 1，就可以直接跳到步骤5。

    步骤2

    然后运行：
    dism.exe /online /enable-feature /featurename:VirtualMachinePlatform /all /norestart
    运行完代码后重启电脑。

    步骤3

    然后下载WSL2 Linux的内核包
  • 适用于 x64 计算机的 WSL2 Linux 内核更新包
    下载完成后双击安装包，然后就会自动安装，等待安装完成后点击Finish即可。

    步骤4

    打开PowerShell，运行下面命令，将 WSL2 设置为默认版本：
    wsl --set-default-version 2

    步骤5

    打开 Microsoft Store搜索WSL下载Linux分发版。根据个人喜好选择下载，我下载的是 Ubuntu 22.04 LTS 这个版本。

至此你已经成功安装WSL,开启你脱发之旅吧！！！！
